Kitan: Java WEB app
=====================================
Java web application based on Spring, Hibernate and Bootstrap.
App for keeping and management commodity turnover in company.

**Technologies**
---------

 - Java 8;
 - Spring 4;
 - Spring Security 4;
 - Hibernate 5;
 - PostgreSQL;
 - Bootstrap;
